HTML常用标签介绍

Posted on 2019-01-05

1.h标签：标题标签.
一共有6种形式，h1~h6，字体的大小权重依次递减，h标签内容都是独占一行。
注意：一个网页通常规范中只允许使用一个h1标签。
2.p标签：段落标签。<p></p>,可以用来放段落文字，内容独占一行。
3.pre标签：格式化文本标签。<pre></pre>
在HTML文档中，无论多少个换行或者空格，最后只会解析成一个空格。使用pre标签包裹的文本会按照原样显示，不会出现这种情况。
pre标签的内容也是独占一行。
pre标签包裹的内容也可以是代码，

显示代码时”<”用”<”代替，”>”用”>”代替
：例如 <p></p>
4.em/i标签:都是用来倾斜字体，都是内容撑开宽度，两种的区别在于em标签的权重大于i标签，通常i标签用的比较多。
<em></em>
<i></i>
5.strong/b标签：都是用来加粗字体，都是内容撑开宽度，strong的强调权重大一些，着重记strong标签。
<strong></strong>
<b></b>
6.span标签:用来包裹文本，也是内容撑开宽度，没有其他的实际意义。
注意：凡是内容撑开宽度的标签，它所包裹的文本中无论有多少个换行和空格，系统都默认解析成\
7.sub/sup标签：不常用，分别用来给一些公式加下标和上标，都是内容撑开宽度，记的话可以看b和p的圆是在上方还是下方，上方就是上标，下方就是下标。
<sub></sub>
<sup></sup>
8.del/ins标签: 给文本加删除线，给文本加下划线，都是内容撑开宽度。
9.a标签：用来添加超链接，是内容撑开宽度。用来跳转到另一个网站。
<a href = “网址”>链接名字</a>，这种用法是站外跳转。
还有一种用法是站内跳转，就是锚点。
<a href = “#锚点名称”></a>
<a name = “锚点名称”></a>
如果在一个比较大的页面浏览时，可能需要不断滑动鼠标滚轮才能找到想要的内容，
我们可以把网页分成若干个部分，每个部分做成一个链接，即所谓的锚点，要想看某个部分的内容，只要鼠标点击对应的锚点即可。
10.img图片标签：是单标签。
src属性是图片的路径 ，路径又分为3种，相对路径，域名路径和磁盘路径。
相对路径是相对于html文档的路径，域名路径是网上图片的地址，磁盘路径是计算机本地的地址。
重点看一下相对路径，
如果图片所在的文件夹是html文档所在文件夹的下级文件夹，那么路径值应该写成”下级文件夹名1/下级文件夹名2/…./图片所在文件夹/图片文件名”即可。(有多少个下级文件夹就写多少个，按级层顺序写)
如果图片所在的文件夹与html文档所在文件夹是同一个文件夹，那么路径值直接写成写成:”所在文件夹名/图片名”即可。
如果图片所在的文件夹是html文档所在文件夹的上级文件夹，那么路径值应该写成:”../../../图片名称”即可，有多少个上级文件夹就写多少个”../“。
再说一下alt属性，alt属性值是一个提示信息，当图片加载失败时就会出现这种提示。
img标签还有一个title属性，就是图片标题。设定好之后，把鼠标放到图片上时就会显示图片的标题信息。
img也有height和width属性，即图片的高度和宽度。如果不给定这两个属性值，那么系统会默认根据原图的大小给定默认值。
